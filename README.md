# Data and Cybersecurity Portfolio Website (C5)


## Tasks Covered
This website documents three completed rooms:

1. **Task 1 –  Social Engineering Password Patterns**  
   Social engineering → credentials → HTTP API access to sensors and actuators (fan/LED).
2. **Task 2 – Wiretap the Secret (packet capture) Forensics**  
   Packet capture → HTTP stream reconstruction → credential + message extraction in plaintext.
3. **Task 4 – Membership Inference**  
   Shadow models + attack model to infer membership on a ResNet18/CIFAR-10 target classifier.

> The **formal write-up** (Executive Summary, vulnerabilities, mitigations, and reflection) is submitted as a PDF via Moodle.
